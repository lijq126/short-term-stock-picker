# A股短线精灵 — Short-Term Stock Picker

> 基于 [WorkBuddy](https://www.codebuddy.cn) 的 A 股智能选股推荐系统 Skill，支持**股票推荐（模式A）**和 **ETF T+0 日内交易推荐（模式B）**两大核心模式。通过实时行情数据、技术面分析和市场情绪评估，提供专业级的短线交易建议。

## 特性

### 模式A：股票推荐
- **四大市场覆盖**：沪深主板、创业板、科创板、北交所，共 8 只候选标的
- **六维评分模型**：涨幅动量 / 量价齐升 / 板块情绪联动 / 技术形态 / 资金流入 / 情绪位置
- **两步筛选机制**：初筛（快速过滤 5~8 只） → 复筛（并行深度扫描 + 六维打分）
- **完整风控体系**：单只仓位上限、总仓位 60%、严格止损 -5%、资金面警告机制

### 模式B：ETF T+0 推荐
- **两大品类覆盖**：跨境 ETF（纳指/日经/恒生科技等） + 商品 ETF（黄金/原油/白银等），共 4 只标的
- **五维评分模型**：波动空间 / 溢价折价 / 流动性 / 趋势动量 / 资金偏好（专为 ETF 日内波段设计）
- **溢价风控**：溢价率 >5% 禁止买入、>3% 减半仓位、溢价风险分级表
- **日内操作策略**：低吸参考价、最佳操作窗口、止盈 +1%~2%/+3%~5% 分批、止损 -1.5% 无条件

### 通用能力
- **实时数据驱动**：通过 NeoData 金融数据接口获取实时行情，禁止使用训练数据编造行情
- **双模式并行**：用户可同时请求股票 + ETF 推荐，一次执行输出两份推荐
- **标准化报告输出**：每次推荐生成结构化 Markdown 报告

## 安装

### 前置条件

1. 已安装 [WorkBuddy](https://www.codebuddy.cn) 桌面端
2. 已配置 **NeoData 金融数据** Skill（数据来源）

### 安装步骤

**方法一：手动安装（推荐）**

1. 克隆或下载本仓库：
   ```bash
   git clone https://github.com/lijq126/short-term-stock-picker.git
   ```

2. 将 `skill/` 目录下的所有文件复制到 WorkBuddy 用户级 Skill 目录：
   - **Windows**: `%USERPROFILE%\.workbuddy\skills\short-term-stock-picker\`
   - **macOS / Linux**: `~/.workbuddy/skills/short-term-stock-picker/`

   目录结构应为：
   ```
   ~/.workbuddy/skills/short-term-stock-picker/
   ├── SKILL.md
   └── references/
       ├── report-template.md
       ├── sentiment-framework.md
       ├── short-term-rules.md
       └── technical-analysis.md
   ```

3. 重启 WorkBuddy，Skill 即可使用

**方法二：通过 WorkBuddy Skill 管理器安装**

1. 在 WorkBuddy 中打开 Skill 管理器
2. 选择"从 GitHub 导入"
3. 输入 `lijq126/short-term-stock-picker`
4. 确认安装

## 执行流程

### 股票推荐流程

```
第一步：查询全市场行情数据（大盘情绪 + 分市场强势股 + 板块资金）
    ↓
第二步：初筛候选池（涨幅/量价/板块三维度快速过滤，每市场 5~8 只）
    ↓
第三步：复筛深度扫描（并行查询个股行情/技术面/资金流向/近期走势）
    ↓
第四步：六维综合评分（涨幅动量20% + 量价齐升20% + 板块情绪20% + 技术形态20% + 资金15% + 情绪5%）
    ↓
第五步：生成推荐报告（标准化 Markdown 格式）
```

### ETF T+0 推荐流程

```
第一步：查询全球关联指标（隔夜美股/美元指数/汇率/大宗商品/VIX）
    ↓
第二步：查询 ETF 行情数据（跨境 ETF 前20 + 商品 ETF 前20 + 技术面）
    ↓
第三步：ETF 初筛（流动性/溢价率/振幅/趋势 四维度过滤）
    ↓
第四步：ETF 五维评分（波动空间25% + 溢价折价25% + 流动性20% + 趋势动量20% + 资金偏好10%）
    ↓
第五步：生成 ETF 推荐报告（含日内操作计划和溢价分析）
```

## 评分模型

### 股票六维评分（模式A）

| 维度 | 权重 | 说明 |
|------|------|------|
| 涨幅动量 | 20% | 今日涨幅、近 3 日涨幅加速度 |
| 量价齐升 | 20% | 当日量比、成交额放大倍数 |
| 板块与情绪联动 | 20% | 板块涨幅 + 情绪热度 |
| 技术形态 | 20% | MA + MACD + KDJ + BOLL + 量价 |
| 资金流入 | 15% | 主力净流入、超大单行为 |
| 情绪位置 | 5% | 连板位置、追高风险 |

### ETF 五维评分（模式B）

| 维度 | 权重 | 说明 |
|------|------|------|
| 波动空间 | 25% | 近 5 日平均振幅，日内做 T 盈利空间 |
| 溢价/折价 | 25% | 跨境 ETF 核心风控指标，溢价越低越安全 |
| 流动性 | 20% | 日成交额，日内交易生命线 |
| 趋势动量 | 20% | 均线排列 + MACD + 与标的指数走势一致性 |
| 资金偏好 | 10% | 近 3 日成交额变化、资金流入方向 |

## 风控规则

### 股票模式（模式A）

| 规则 | 说明 |
|------|------|
| 候选池 8 只，精选 ≤ 5 只 | 优先选综合评分最高的前 3~5 只 |
| 单只仓位上限 | 主板 20% / 创业板 15% / 科创板 15% / 北交所 10% |
| 总仓位上限 | 60% |
| 止损 | 买入价 -5% 无条件执行 |
| 止盈 | 盈利 ≥ 8% 减半仓 |
| 大盘保护 | 当日跌幅 >2% 停止新开仓 |

### ETF T+0 模式（模式B）

| 规则 | 说明 |
|------|------|
| 单只 ETF 仓位上限 | ≤15% |
| ETF 总仓位上限 | ≤30%（含所有跨境 + 商品 ETF） |
| 溢价禁买线 | 溢价率 >5% 禁止买入（高溢价崩盘风险） |
| 溢价减仓线 | 溢价率 >3% 仓位减半（≤7.5%） |
| 止损 | -1.5% 无条件执行（日内可平仓，无理由扛单） |
| 止盈 | +1% 减仓 50%，+2% 再减 25% |
| 不追涨 | 现价涨幅 >2% 时不新建仓 |
| 尾盘禁买 | 14:55 后禁止新建仓 |

## 协同使用

本 Skill 可与 [multi-agent-investment](https://github.com/lijq126/multi-agent-investment)（多智能体投研分析系统）协同使用，实现：

| 场景 | 说明 |
|------|------|
| 防雷过滤 | 短线精灵推荐后，多智能体投研做基本面排雷 |
| 深度研报 | 短线精灵筛选后，投研系统生成完整分析报告 |
| 板块持续性 | 短线精灵看当日板块强度，投研系统分析板块中长期逻辑 |
| 资金面诊断 | 短线精灵看当日资金流向，投研系统分析机构持仓变化 |
| 矛盾检测 | 短线精灵技术面看多 vs 投研系统基本面看空时发出警告 |

详细协同场景说明见 [SYNERGY.md](./SYNERGY.md)。
