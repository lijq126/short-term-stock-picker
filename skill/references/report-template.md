# 短线股票推荐报告 — 标准模板

每次执行短线选股任务时，按以下格式输出报告。

**六维评分说明列的填写指引**：
- **主板**：侧重"板块龙头地位+资金持续流入+均线多头排列"
- **创业板**：侧重"20cm弹性空间+放量突破信号+短线资金活跃度"
- **科创板**：侧重"科技赛道主线地位+机构资金行为+研发/估值支撑"
- **北交所**：侧重"流动性充裕度（成交额）+板块联动+价格低位弹性"

**特殊标注规则**：
- **⚠️ 涨停观察标的**：涨停封板中无流动性的板块龙头，不计入正式推荐，标注"等回踩/开板后再介入"
- **⚠️ 资金面警告**：资金面评分≤3分（主力净流出），仍可推荐但风险提示中必须醒目标注

**报告头部必须包含**：
- 报告日期与生成时间
- 数据截止时间（盘中实时/收盘数据/最近交易日收盘）
- 数据来源标注
