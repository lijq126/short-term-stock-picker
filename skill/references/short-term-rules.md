# 短线交易核心规则与禁忌

## 铁律总纲

> **短线交易的核心不是"买什么"，而是"什么时候买、什么时候卖、什么时候认错"。**
> 再好的标的，买卖点错了就是亏损。再差的标的，买卖点对了也能盈利。

---

## 一、短线选股标准（必须同时满足）

### 基本条件（不满足直接淘汰）

- [ ] 上市时间 > 3个月（非次新股）
- [ ] 非 ST / *ST / S 股（含退市风险警示）
- [ ] 今日成交额 > 1亿元（流动性要求）
- [ ] 非涨停封板中（涨停板无流动性无法买入）
- [ ] 近期1周无解禁压力

### 技术条件（至少满足3条）

- [ ] 价格站上 MA5 均线
- [ ] MA5 > MA10（均线多头）
- [ ] MACD 金叉（ DIF 上穿 DEA）
- [ ] KDJ 金叉（ K 上穿 D）
- [ ] 价格在布林带中轨上方
- [ ] 量比 > 1.5

### 资金条件（必须满足）

- [ ] 今日主力净流入为正
- [ ] 超大单净流入 > 大单净流入（机构资金主导）

> **⚠️ 资金面降级处理**：若主力净流出但其他维度极强（如板块龙头+技术形态完美+情绪高涨），可降级为"资金面警告标的"参与推荐，但必须：
> 1. 资金面维度评分≤3分
> 2. 综合评分下调0.5分作为风险补偿
> 3. 报告中标注"⚠️ 资金面警告"，风险提示中醒目提示
> 4. 建议仓位减半（如主板标的由≤20%降至≤10%）

### 情绪条件（必须满足）

- [ ] 所属板块今日涨幅 > 0.5%

---

## 二、买入原则（5条铁律）

### 买入时机

1. **早盘买入**：短线最佳买入时间是 **9:30~10:00**（开盘后30分钟看清方向）
   - 忌：集合竞价高位追入（容易被套）
   - 宜：开盘后股价回踩均线/均价线时买入

2. **分批建仓**：不一次性买满计划仓位，分2次买入
   - 第一次：信号确认后买入计划仓位的**50%**
   - 第二次：回调确认不破买入价时买入计划仓位的**30%**
   - 保留20%应变空间
   - **注**：计划仓位指为该标的分配的仓位上限（如主板标的计划仓位20%，则第一次买10%、第二次买6%）

3. **只买主线**：每天只做市场最强主线，不做跟风、不做杂毛

4. **首板优先**：同一板块内，**优先选择今日首板的个股**（补涨风险低）

5. **单只仓位上限**：
   - 主板标的：单只 ≤ 总仓位20%
   - 创业板标的：单只 ≤ 总仓位15%
   - 科创板标的：单只 ≤ 总仓位15%
   - 北交所标的：单只 ≤ 总仓位10%

---

## 三、卖出原则（3条铁律）

> **会买的是徒弟，会卖的是师父。**

### 止盈策略

| 情况 | 操作 |
|------|------|
| 买入后当日涨停 | **持有到尾盘**，若炸板立即卖出 |
| 买入后涨幅 > 8% | **减仓50%**，锁住利润 |
| 买入后涨幅 5%~8% | 持有至次日，根据开盘情况决定 |
| 买入后涨幅 < 3% | **次日冲高卖出**，不恋战 |

### 止损策略

> **止损是短线交易的生命线。没有止损的短线，等于赌博。**

| 情况 | 操作 |
|------|------|
| 买入后下跌 > 5% | **无条件止损**，不要幻想反弹 |
| 买入后跌破 MA5 | **次日开盘止损** |
| 大盘当日跌幅 > 2% | **清仓观望**，不做新开仓 |
| 个股炸板（涨停打开） | **立即卖出**，不抱幻想 |

## 四、仓位管理

### 大盘环境仓位对照表

| 大盘状态 | 建议仓位 | 操作策略 |
|---------|---------|---------|
| 大盘极强（涨幅>1.5%，涨停>80家） | 50%~60% | 积极做多，可追龙头 |
| 大盘偏强（涨幅0.5~1.5%） | 40%~60% | 聚焦主线，控制仓位 |
| 大盘中性（涨跌互现） | 20%~40% | 精选个股，快进快出 |
| 大盘偏弱（跌幅-0.5~-1.5%） | 0%~20% | 轻仓或空仓 |
| 大盘极弱（跌幅>1.5%） | **0%** | **全面空仓，禁止开仓** |

> **注意**：无论大盘多强，总仓位**永远不超过60%**（留40%现金应对极端情况）。

### 总仓位控制

- 同时持仓不超过 **5只**（推荐8只为候选池，实际买入从中精选≤5只，优先选综合评分最高的）
- 总仓位不超过 **60%**（留40%现金应对极端情况）
- 科创板标的总仓位不超过 **30%**
- 北交所标的总仓位不超过 **10%**

## 五、禁忌清单（绝对不要做）

### 绝对禁区

1. **❌ 不要追涨3板以上的妖股**
   - 3板以后是纯情绪博弈，一旦炸板往往直接跌停
   - 历史数据：连续3板后，第4板炸板率>60%

2. **❌ 不要在尾盘最后10分钟买入**
   - 尾盘买入无法观察当日走势，风险放大
   - 最佳买入窗口：9:30~10:00 和 13:00~13:30

3. **❌ 不要在集合竞价时追高**
   - 9:15~9:25的集合竞价是主力诱多的高发时段
   - 正确做法：9:30分开盘确认方向后再操作

4. **❌ 不要在亏损时加仓摊低成本**
   - 短线交易的核心是"截断亏损"，加仓只会让亏损扩大
   - 亏损>5%立即止损，不幻想，不扛单

5. **❌ 不要全仓单只**
   - 全仓单只的胜率<40%，一旦失败损失惨重
   - 即便非常看好，单只仓位不超过20%

6. **❌ 不要在跌停板上买入**
   - 跌停板买入了"接飞刀"，次日往往继续跌
   - 历史数据：跌停次日继续跌停的概率>40%

7. **❌ 不要连续交易（梭哈心理）**
   - 每日新开仓不超过2只
   - 连续3日亏损后，强制休息1周

## 六、自检清单（每日复盘前必读）

### 买入前检查

- [ ] 大盘今日跌幅是否超过2%？（若是，禁止开仓）
- [ ] 该标的是否满足6条基本条件？
- [ ] 该标的今日是否有板块联动支撑？
- [ ] 买入价格是否在合理区间（<3%追高）？
- [ ] 止损位是否明确（-5%）？

### 持仓中检查

- [ ] 当日是否触及止盈/止损位？
- [ ] 板块是否出现退潮迹象（板块整体回落）？
- [ ] 大盘是否出现系统性风险？

### 卖出后检查

- [ ] 卖出理由是什么？（止盈/止损/板块退潮/其他）
- [ ] 本次交易盈亏情况
- [ ] 是否有需要改进的地方

## 七、ETF T+0 交易规则

> **ETF T+0 是一把双刃剑——当日买卖意味着你可以随时止损，但也意味着亏损可以在几小时内放大。纪律比股票更重要。**

### ETF T+0 选品标准（必须同时满足）

#### 基本条件（不满足直接淘汰）

- [ ] 日成交额 > 5000万（流动性要求）
- [ ] 跨境ETF溢价率 < 5%（溢价率>5%直接排除，>3%减仓）
- [ ] 非上市不足1个月的次新ETF
- [ ] 非分级基金（分级基金已基本退市，但仍需排除）

#### 波动条件（至少满足2条）

- [ ] 近5日平均振幅 > 1%
- [ ] 当日振幅 > 1%
- [ ] 5日均线方向向上（顺趋势操作）
- [ ] MACD方向向上

### ETF T+0 买入原则

#### 买入时机

1. **开盘观望再操作**：ETF日内操作的核心是**看清方向再动手**
   - 9:30~9:45：**观望期**，观察海外隔夜数据对今日开盘的影响方向
   - 9:45~10:00：**操作窗口1**，方向明确后低吸
   - 13:00~14:00：**操作窗口2**，午后反弹期（商品ETF 14:00~14:30 波动常较大）
   - 14:55之后：**禁止新建仓**（避免无法当日平仓的风险）

2. **低吸为主，不追涨**：
   - 买入价格应低于当日均价线（分时图的VWAP线）
   - 现价涨幅>2%时不建议新建仓（ETF弹性小于股票，追涨性价比低）
   - 理想买入点：日内回调至MA5或均价线附近

3. **分批建仓**：
   - 第一次：信号确认后买入计划仓位的 **60%**
   - 第二次：回调不破买入价时买入计划仓位的 **30%**
   - 保留10%应变空间

#### 单只仓位上限

| 类型 | 单只上限 | 总仓位上限 |
|------|---------|-----------|
| 跨境ETF | ≤15% | ≤30%（含所有ETF） |
| 商品ETF | ≤15% | ≤30%（含所有ETF） |

### ETF T+0 止盈策略

> **ETF盈利的本质是"小赚多次"而非"一把大赚"。每次赚1%~2%，积少成多。**

| 情况 | 操作 |
|------|------|
| 买入后涨幅 > 2% | **减仓75%**，留25%底仓观察趋势 |
| 买入后涨幅 1%~2% | **减仓50%**，锁定日内利润 |
| 买入后涨幅 < 1% | 持有至操作窗口2（午后），观察是否有第二波 |
| 趋势明确做多 | 可留20%底仓过夜，次日冲高卖出 |

### ETF T+0 止损策略

> **ETF止损比股票更严格——因为日内可平仓，没有理由扛单。**

| 情况 | 操作 |
|------|------|
| 买入后下跌 > 1.5% | **无条件止损**，不幻想反弹 |
| 买入后跌破日内均价线且无反弹 | **立即止损** |
| 海外市场突发利空（盘中） | **立即止损**，跨境ETF反应极快 |
| 临近收盘仍亏损 | **尾盘清仓**，不留过夜风险 |

### ETF T+0 溢价风险控制

> **溢价是跨境ETF最大的隐性风险**。溢价率=（ETF市价 - 实际净值）/ 实际净值。高溢价意味着你买贵了。

| 溢价率范围 | 风险等级 | 操作建议 |
|-----------|---------|---------|
| < 0%（折价） | 最优 | 可以买入，折价 = 安全边际 |
| 0~1% | 低风险 | 正常买入 |
| 1~3% | 中风险 | 正常买入，但需关注溢价变化趋势 |
| 3~5% | 高风险 | 减半仓位（≤7.5%），报告中标注"⚠️溢价警告" |
| > 5% | 极高危 | **禁止买入** |

> **历史教训**：2024年1月，日经ETF溢价率一度超过20%，随后连续暴跌，追高者损失惨重。

### ETF T+0 禁忌清单

1. **❌ 不要买高溢价ETF**（溢价率>5%，等于主动送钱）
2. **❌ 不要重仓单只ETF**（单只≤15%，ETF日内波动可能超预期）
3. **❌ 不要在尾盘最后5分钟建仓**（无法当日平仓）
4. **❌ 不要逆趋势操作**（海外指数跌你做多跨境ETF，等于逆风行驶）
5. **❌ 不要把ETF当股票炒**（ETF不会涨停，追涨的意义远小于股票）
6. **❌ 不要忽视汇率影响**（跨境ETF的涨跌 = 海外资产涨跌 × 汇率变动）

### ETF T+0 自检清单

#### 买入前检查

- [ ] 溢价率是否 < 5%？（跨境ETF）
- [ ] 日成交额是否 > 5000万？
- [ ] 当前是否处于操作窗口（9:45~10:00 / 13:00~14:00）？
- [ ] 对应海外指数/商品价格趋势是否向上？
- [ ] 止损位是否明确（-1.5%）？

#### 持仓中检查

- [ ] 是否触及 +2% 止盈位？（是则减仓75%）
- [ ] 是否触及 -1.5% 止损位？（是则无条件止损）
- [ ] 海外市场是否有突发消息？（跨境ETF）
- [ ] 临近14:55是否仍持仓？（是则考虑清仓不过夜）
