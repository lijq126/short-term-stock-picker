# 技术面分析框架 — 短线选股专用

## 核心原则

短线技术分析只看**三个核心信号**：**均线多头发散**、**MACD底部金叉**、**量价齐升**。不追求完美指标，只追求信号共振。

---

## 一、均线系统（MA）

### 短线最重要的3条均线

| 均线 | 周期 | 短线含义 |
|------|------|---------|
| MA5 | 5日 | 短期成本线，**生命线**，跌破需警惕 |
| MA10 | 10日 | 短期趋势线，线上做多，线下休息 |
| MA20 | 20日 | 中期趋势线，**多空分界线** |

### 均线形态评分

| 形态 | 信号强度 | 评分(10分) |
|------|---------|-----------|
| MA5 > MA10 > MA20 多头排列 | 最强 | 9~10 |
| MA5 上穿 MA10（金叉），价格在 MA20 上方 | 强势 | 7~8 |
| MA5 在 MA10 上方横向整理 | 中性偏强 | 5~6 |
| 价格在 MA5 附近波动 | 中性 | 3~4 |
| MA5 < MA10，价格在均线下方 | 弱势，不选 | 0~2 |

---

## 二、MACD 指标

### 参数设置（标准短线配置）

- DIF: 12日EMA - 26日EMA
- DEA: 9日DIF的EMA
- MACD柱 = (DIF - DEA) × 2

### 四大经典形态与评分

| 形态 | 含义 | 评分(10分) |
|------|------|-----------|
| **MACD 金叉**（DIF上穿DEA）+ 绿柱转红柱 | 最强买入信号 | 9~10 |
| MACD 在0轴上方运行 + 红柱收缩 | 强势整理，等待 | 7~8 |
| MACD 在0轴下方运行 + 绿柱持续收缩 | 底部蓄力 | 5~6 |
| DIF 与 DEA 反复交叉（缠绕） | 横盘，不选 | 2~4 |
| MACD 在0轴下方 + 红柱变绿柱 | 弱势，不选 | 0~2 |

### 高级形态：MACD 底背离（重要！）

> 当股价创出新低，但MACD没有创新低 → **强烈底部信号**，往往是大涨前兆

---

## 三、KDJ 指标

### 参数设置

- n=9, m1=3, m2=3（J线波动最大，最灵敏）

### KDJ 形态评分

| 形态 | 信号 | 评分(10分) |
|------|------|-----------|
| K线上穿D线（金叉）+ K<30 | 超卖区金叉，**黄金买点** | 9~10 |
| K线上穿D线（金叉）+ K在30~70 | 中性金叉 | 6~7 |
| KDJ 在80以上高位钝化 | 超买，谨慎追高 | 3~4 |
| K线下穿D线（死叉）+ K>70 | 高位死叉，不选 | 0~2 |

### KDJ 高级形态：底背离

> 股价创新低，KDJ没有创新低 → 底部信号

---

## 四、布林带（BOLL）

### 参数设置

- 中轨 = 20日均线
- 上轨 = 中轨 + 2倍标准差
- 下轨 = 中轨 - 2倍标准差

### 布林带短线用法

| 形态 | 信号 | 评分(10分) |
|------|------|-----------|
| 价格从下轨附近反弹，穿越中轨 | 底部启动信号 | 8~9 |
| 价格在中轨与上轨之间运行 | 上升通道持有 | 6~7 |
| 价格突破上轨（张口） | 可能加速，注意追高风险 | 4~5 |
| 价格从上轨附近回落 | 短线卖点 | 3~4 |
| 价格持续在下轨下方运行 | 弱势，不选 | 0~2 |

---

## 五、量价关系（最核心！）

### 短线量价口诀

> **"量为价先，无量不涨，天量见天价"**

### 量价评分体系

| 量价形态 | 信号 | 评分(10分) |
|---------|------|-----------|
| 放量上涨（量比>2）+ 涨幅>5% | 最强，机构/主力进场 | 9~10 |
| 温和放量（量比1.5~2）+ 稳步上涨 | 健康，可参与 | 7~8 |
| 缩量上涨（量比<0.8） | 主力控盘，但上涨乏力 | 4~5 |
| 放量下跌（量比>2）+ 跌幅>5% | 主力出货，不选 | 0~2 |
| 平量整理（量比0.8~1.2）| 横盘整理，等方向 | 3~4 |

### 量比指标（Volume Ratio）

- 量比 = 当日每分钟平均成交量 / 过去5日每分钟平均成交量
- 量比>2：显著放量
- 量比1.5~2：温和放量
- 量比<0.8：缩量

---

## 六、综合技术评分计算

```
技术面总分 = 均线得分×30% + MACD得分×25% + KDJ得分×20% + 布林带得分×15% + 量价得分×10%
```

| 总分区间 | 信号 | 操作建议 |
|---------|------|---------|
| 8.0~10 | 极强 | 积极买入 |
| 6.5~8.0 | 强势 | 买入 |
| 5.0~6.5 | 中性 | 轻仓或观望 |
| 3.5~5.0 | 弱势 | 不选 |
| <3.5 | 极弱 | 规避 |

---

## 七、常见短线技术陷阱（需规避）

1. **高位放量十字星** → 主力出货信号，次日容易低开
2. **KDJ高位钝化后死叉** → 快速下跌，往往跌幅>10%
3. **涨停板反复炸板** → 封板不稳，次日低开概率高
4. **缩量一字板后T字板** → 连续一字后开板慎接
5. **跌破MA5且3日内无法收复** → 短线趋势走坏，止损信号
