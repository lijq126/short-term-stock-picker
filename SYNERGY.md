# 短线精灵 x 多智能体投研系统 协同场景

> 短线精灵（short-term-stock-picker）可以与[多智能体投研分析框架](https://github.com/lijq126/multi-agent-investment)（multi-agent-investment）深度协同，一个管**短线快选**（量化驱动、1-3日），一个管**深度投研**（分析驱动、中长期），互补性很强。

---

## 系统能力边界

| 能力维度 | 短线精灵 | 多智能体投研 |
|---------|---------|------------|
| 分析周期 | 1-3日短线 | 中长期 |
| 驱动方式 | 量化评分（六维模型） | 多角色分析+多空辩论 |
| 数据来源 | 实时行情+技术面+资金面 | 财报+行业+情绪+政策 |
| 输出形式 | 选股组合+操作计划 | 深度研报+投资评级 |
| 强项 | 快速筛选、技术信号 | 深度验证、风险评估 |
| 弱项 | 基本面覆盖浅 | 缺少短线技术面 |

---

## 协同场景

### 场景一：短线快筛 + 深度验证（防雷过滤器）

**流程**：

```
短线精灵选出8只候选
    ↓
对综合评分 Top3 标的触发多智能体基本面+风险分析师快速诊断
    ↓
过滤掉有雷的标的
```

**价值**：短线精灵只看技术面和资金面，不看财报质量。一只票技术形态完美但可能刚发布业绩暴雷预警，多智能体的基本面分析师能立刻发现。相当于给短线选股加了一层**防雷网**。

**触发示例**：
```
帮我选短线，并对推荐的前3只做基本面排雷
```

---

### 场景二：精选标的深度研报（短线版投研报告）

**流程**：

```
短线精灵推荐 Top1
    ↓
触发多智能体完整4角色分析 + 多空辩论
    ↓
输出混合研报：短线操作计划 + 中长期逻辑
```

**价值**：短线精灵的报告偏"什么时候买卖"，多智能体偏"值不值得买"。合在一起就是一份完整的交易决策：**短线技术信号 + 中长期基本面支撑**。

**触发示例**：
```
对今天短线推荐的Top1做深度研报
```

---

### 场景三：板块主线深度研究（判断持续性）

**流程**：

```
短线精灵识别当日主线板块（如"小金属 +6.82%"）
    ↓
多智能体行业分析师做该板块深度研究 + 多空辩论
    ↓
判断是一日游还是持续性主线
```

**价值**：短线精灵能告诉你"今天什么板块在涨"，但无法判断"还能涨多久"。行业分析师能评估产业链逻辑、政策周期、估值水位，帮用户判断**要不要追这个主线**。

**触发示例**：
```
今天小金属板块大涨，帮我研究这个板块的持续性
```

---

### 场景四：资金面警告标的深度诊断

**流程**：

```
短线精灵标记 "资金面警告" 的标的（主力净流出但技术面强）
    ↓
多智能体风险分析师 + 情绪分析师专项诊断
    ↓
判断主力流出是"正常洗盘"还是"机构出逃"
```

**价值**：短线精灵的六维评分无法区分"洗盘"和"出货"，但多智能体可以结合大宗交易数据、龙虎榜席位、近期公告等综合判断。

**触发示例**：
```
XXX 主力净流出但技术面很好，帮我诊断一下
```

---

### 场景五：短线 vs 中长期矛盾检测（最有趣的场景）

**流程**：

```
短线精灵推荐买入 + 多智能体基本面看空
    ↓
自动识别矛盾
    ↓
输出"短线炒作 vs 价值陷阱"矛盾分析报告
    ↓
给出更安全的操作建议
```

**价值**：这种情况在A股特别常见——一只亏损公司因为题材炒作连板。短线精灵会推荐（技术面强），但多智能体会看空（基本面差）。两者矛盾时，系统应该**自动提醒风险**，让用户知道自己是在"炒题材"而不是"投价值"。

**触发示例**：
```
帮我选短线，同时检查推荐标的中有没有基本面雷
```

## 使用前提

要使用协同场景，需要同时安装两个 Skill：

1. **短线精灵**：本仓库，安装方法见 [README.md](./README.md)
2. **多智能体投研系统**：另见 [multi-agent-investment](https://github.com/lijq126/multi-agent-investment) 仓库

两个 Skill 安装后，在对话中自然语言描述即可触发协同场景，系统会自动识别需要调用哪个 Skill。
